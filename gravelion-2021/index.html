ГОВЕРЛА

Четвертий Gravelion проходив біля села Говерла.
Івент вразив своєю маштабністю – було більше 200 учасників, і здається ще стільки ж глядачів!

Для такої кількості людей вже не так просто знайти поляну під кемп, тому ми зробили навіть два кемпи!
“Верхній кемп” для фану, “нижній” – для душі.

Окремим сюрпризом для учасників був маршрут – він збудований по свіжим гравійним дорогам,  хоча без кантріліонних ділянок теж не обійшлось 🙂

Маршрут, як завжди, поділений на ділянки. Цього року їх було три! Між двома основними ділянками, ми додали одну додаткову, з найкращими краєвидами на маршруті, яку учасники проїхали у довільному темпі, не турбуючись про залік по часу.
Фішка маршуту у тому, що він наполовину пролягає по нещодавно прокладених дорогах (у 2015-2018 роках).
